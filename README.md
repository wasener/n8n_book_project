# n8n_book_project

本書依據內容難度與 n8n 官方認證體系，分為兩冊：

【第一冊】n8n 自動化工作流程：基礎篇（共 9 章）
涵蓋技術基礎知識、n8n 基礎操作、AI 節點入門與豐富的實戰專題，完整對應 n8n 官方 Level 1 認證範圍，並預先介紹 AI 自動化基礎概念。適合初學者從零開始建立完整的自動化基礎能力。

【第二冊】n8n 自動化工作流程：進階篇（共 6 章）

涵蓋進階資料處理、外部服務整合、AI Agent 進階應用與企業級部署，對應 n8n 官方 Level 2 認證範圍。適合進階使用者深化技能與企業部署。
